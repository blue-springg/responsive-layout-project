Basic Media Queries

In this assignment you will add two media queries to the css file.

  • The first media query should trigger at 772px and change the layout of the boxes to be 45% of the viewport width. The images should have a maximum height of 30% of the viewport height.
  • The second media query should trigger at 998px and change the layout of the boxes to be 30% of the viewport width.
